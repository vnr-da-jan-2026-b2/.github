# Data Analytics Training (VNR 2024-2028)

This is the code repository of the project based industry-aligned training program for the **Data Analyst** roles.

In building these Data powered products, students exercised their skills across various tools of data analysis ( `Excel`, `SQL`, `Python` and `Story-telling` ) by working in a collaborative setting of Agile squad/teams.

This program is run in collaboration with **VNR VJIET** (https://vnrvjiet.ac.in/) and **PSNM Innovations** (https://psnm.in/). 

If you would like to be part of similar future training programs, please drop us an email at `hello@psnm.in`.
